CNN - Rede Neural Covolucional.

O objetivo do projeto é de ensinar e retrabalhar o ML, de modo que ele aumente sua precisão com épocas de aprendizado e detalhamento, para que possa classificar peças de metal. O banco de dados utilzado foi extraídodo Kaggle com o link fornecido abaixo:

"https://www.kaggle.com/datasets/ravirajsinh45/real-life-industrial-dataset-of-casting-product/data"

O que foi utilizado para o projeto CNN: 
- Feito em markdown;
- Utilização do Transfer Learning;
- Congelamento de camadas para ajuste de fine-tuning;
- Piloto para a implementação do GRAD-CAM; 
- Utilização do MobileNetv2, rede pré treinada no ImageNet.

Para mais, o relatório do projeto se encontra na última célula do projeto.
